# FreeStar4k-Web
<img src="https://freestar.9dcrew.org/scr.jpg">
FreeStar4k is a high-accuracy simulation of the WeatherSTAR 4000, a computer system designed fby The Weather Channel for generating on screen graphics for showing the weather.
<br><br>

This fork's main goal is for support with ```pygbag```, however it also acts as a continuation of the original project.

## Setup and Compiliing
Install all the dependences with ```pip install -r requirements.txt```, or if you are on an externally managed enviroment track the packages down in your os's package manager.

Install `pygbag` with pip, and in the root of the project run `python3 -m pygbag ./`
<br>The project will compile and automatically start a server at `http://localhost:8000`.
<br>Alternatively, run `python3 main.py` in the project root to run locally.
<br>Currently this is worse then using the online version as manual location input is not supported, however it is coming soon.
